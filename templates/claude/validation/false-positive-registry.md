## Usual suspects (tick the ones you've caught yourself on)
- [ ] Compliments ("that's cool", "great idea", "you should build it")
- [ ] Hypothetical enthusiasm ("I would definitely use / pay for that")
- [ ] Vanity metrics (signups, waitlist, likes) with no activation
- [ ] Warm-intro bias (only friends/network said yes)
- [ ] Feature requests read as commitment ("add X and I'm in" — usually they're not)
- [ ] Survey scores without corresponding behaviour

## Standing rule
- **A signal counts as validation only if it is:** past-tense behaviour, costs
  the customer something (money/time/effort), and is repeatable — not a one-off.
