# Example: Login Flow

**URL:** https://practicetestautomation.com/practice-test-login/

## Steps

1. Navigate to the login page
2. Enter username "student" in the Username field
3. Enter password "Password123" in the Password field
4. Click the "Submit" button
5. Verify the page shows "Congratulations" or "successfully logged in" message

## Expected

- Login page loads without errors
- Username and password fields accept input
- Submit button is clickable
- After login, a success message is displayed
- Page URL changes to indicate successful authentication
